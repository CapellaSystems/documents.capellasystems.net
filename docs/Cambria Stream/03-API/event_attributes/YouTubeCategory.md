# YouTubeCategory

## XML Element
Event

## XML Attribute
YouTubeCategory

## Description
Specifies the YouTube category for the event.

|----------------|--------------------------|

| `Comedy`       | Comedy                   |
| `Education`    | Education                |
| `Entertainment`| Entertainment            |
| `Film`         | Film & Animation         |
| `Games`        | Gaming                   |
| `Howto`        | Howto & Style            |
| `Music`        | Music                    |
| `News`         | News & Politics          |
| `Nonprofit`    | Nonprofits & Activism    |
| `People`       | People & Blogs           |
| `Animals`      | Pets & Animals           |
| `Tech`         | Science & Technology     |
| `Sports`       | Sports                   |
| `Travel`       | Travel & Events          |
