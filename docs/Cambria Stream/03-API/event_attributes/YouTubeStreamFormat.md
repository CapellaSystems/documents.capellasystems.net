# YouTubeStreamFormat

## XML Element
Event

## XML Attribute
YouTubeStreamFormat

## Description
Specifies the YouTube stream format available for an event.

## Values

| Value         | Description    |
|---------------|----------------|
| `240p`        | 240p           |
| `360p`        | 360p           |
| `480p`        | 480p           |
| `720p`        | 720p           |
| `720p_hfr`    | 720p (60fps)   |
| `1080p`       | 1080p          |
| `1080p_hfr`   | 1080p (60fps)  |
| `1440p`       | 1440p          |
| `1440p_hfr`   | 1440p (60fps)  |
